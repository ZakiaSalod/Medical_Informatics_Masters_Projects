Steps for successfully connecting to database server and opening the .JAR file :
by: Zakia Salod | 2016
====================================================================================================
1) Double-click on "usbwebserver.exe" under Usbwebserver/usbwebserver.exe
   This will open-up the USBWebserver window
2) On this window, click on "General". Then double-click on "PHPMyAdmin"
   A browser session with phpMyAdmin will open-up.   
3) Login :
   Username: root
   Password: usbw
   Click on "Go"
   This will open-up a view of all databases.
   Database specifically for this Assignment 02 is:
   medassign03
4) The INFT8F2H2_Assign04_ZakiaS.jar file can then be opened-up on your Eclipse session and run.
   Alternatively, the "INFT8F2H2_Assign04_ZakiaS" file may be opened in your Eclipse session and run.
